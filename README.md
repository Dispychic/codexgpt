## Publication GitHub Pages

1. Envoyer `index.html`, `README.md` et le dossier `assets-web` dans un repository GitHub.
2. Aller dans `Settings` > `Pages`.
3. Choisir `Deploy from a branch`, branche `main`, dossier `/root`.
4. Attendre que GitHub donne l'URL du site.
